# LyricsFinder [![Build Status](https://travis-ci.org/dnlR/lyrics_finder.svg?branch=master)](https://travis-ci.org/dnlR/lyrics_finder) [![Coverage Status](https://img.shields.io/coveralls/dnlR/lyrics_finder.svg)](https://coveralls.io/r/dnlR/lyrics_finder?branch=master)

Simple library to search for song lyrics 

## Installation

Add this line to your application's Gemfile:

    gem 'lyrics_finder'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lyrics_finder
